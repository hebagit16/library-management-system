# Library Management System

This project is part of a Software Testing course.

## Project Description
A simple Library Management System that allows:
- Adding and searching books
- Managing users
- Borrowing and returning books
- Viewing active loans

## Technologies
- Java
- Spring Boot
- REST API
- Selenium WebDriver
- Postman
- JUnit
- Jira

## Testing Scope
- Manual testing (test plan & test cases)
- API testing using Postman
- UI testing using Selenium
- Static testing (code review)
